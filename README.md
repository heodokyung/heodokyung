## 이런 개발을 좋아합니다

<table>
  <tr>
    <td width="50%" valign="top">
      <h3>🎯 사용하기 쉬운 화면</h3>
      <p>화려한 효과보다, 사용자가 자연스럽게 이해할 수 있는 흐름을 더 중요하게 생각합니다.</p>
    </td>
    <td width="50%" valign="top">
      <h3>♿ 기본을 지키는 개발</h3>
      <p>시맨틱 마크업, 키보드 접근성, 명확한 상태 표현처럼 작은 기본기를 놓치지 않으려고 합니다.</p>
    </td>
  </tr>
  <tr>
    <td width="50%" valign="top">
      <h3>🧩 다시 쓰기 좋은 구조</h3>

    </td>
    <td width="50%" valign="top">
      <h3>🛠 오래 가는 코드</h3>

  </tr>
