### Hi there 👋
반갑습니다. 
저는 프론트엔드에 관심이 많은 UI개발자 입니다.   
웹표준과 웹접근성을 준수하고 저의 코딩컨벤션을 만들어서 마크업을 구현합니다.  
코드의 재사용과 범용적인 코드를 작성하려고 노력하며 프론트에 관심이 많습니다.

리액트(React)와 뷰(Vue)를 좋아하고 부족하고 모르는 부분은 노력하고 배우면서 성장하겠습니다.  
감사합니다.
